# DONG

This project was started to create an app to support the drinking game "DONG".

## Getting started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

## Compiling the app

### Local

```bash
eas build --platform android --local --profile development
```

### Remote

```bash
eas build --platform android --profile development
```
